# React Application Template

Quick application template setup with [React](https://reactjs.net/), [Bootstrap](https://getbootstrap.com/), [Babel](https://babeljs.io/), [Jest](https://facebook.github.io/jest/), [ESLint](https://eslint.org/), and the [Parcel](https://parceljs.org/) bundler.

## Usage

Download or clone this repo then install the dependencies.
Since you don't need the git history, just downloading the repository with the download button is suggested.

```bash
# Install dependencies
yarn

# Start the dev server
yarn start

# Build application
yarn build
```

## Linting and Unit Tests

```bash
# Run ESLint then Jest
yarn test

# Run only ESLint
yarn lint

# Run only Jest
yarn jest
```
